# FUTURE_DS_02  
# 📊 Customer Retention & Churn Analysis  

## 📌 Objective  

The objective of this project is to analyze customer data to identify churn patterns, retention trends, and factors affecting customer behavior. The goal is to reduce customer loss and improve retention strategies.  

---

## 🛠️ Tools Used  

* MS Excel  
* Power BI  

---

## 📂 Dataset  

* Customer dataset (subscription-based data)  
* Includes customer activity, usage patterns, and subscription details  

---

## 🧹 Data Preparation  

* Cleaned missing and inconsistent data  
* Organized dataset for analysis  
* Prepared data for visualization  

---

## 📊 Analysis Performed  

* 📉 Churn Rate Analysis  
* 👥 Customer Retention Analysis  
* 📅 Cohort / Time-based Analysis  
* 📊 Customer behavior patterns  

---

## 💡 Key Insights  

* Some customers churn after a short period of inactivity  
* Retention is higher for consistent users  
* Certain patterns indicate higher churn risk  
* Customer engagement plays a key role in retention  

---

## 📸 Project Output  

Visualizations and dashboards were created to show:  
* Churn trends  
* Retention patterns  
* Customer behavior insights  

---

## 🚀 Outcome  

This project helped in understanding customer behavior and how data can be used to improve retention strategies.  

---

## 📢 Conclusion  

Customer retention is critical for business growth. By analyzing churn patterns, businesses can take proactive steps to reduce customer loss.
